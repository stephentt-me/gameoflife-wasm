# Game Of Life - Wasm

An implementation of [Game Of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life), written in Go. Support WASM.

## Prerequisites

* Go >= 1.11
* A morden browser with WebAssembly support.
